# Social Media Content Automation

An AI-powered n8n workflow that automates content creation and social media publishing from news articles.

## Features

* Monitors Google Sheets for new article entries
* Summarizes articles using Google Gemini
* Generates LinkedIn-ready content
* Generates X (Twitter) posts
* Automates social media publishing

## Workflow

Google Sheets → Article Summarization → Content Generation → Social Media Publishing

## Tech Stack

* n8n
* Google Sheets
* Google Gemini
* LinkedIn API
* X (Twitter) API

## Use Cases

* Content marketing automation
* News curation
* Social media management
* AI-assisted content creation

## Getting Started

1. Import the workflow into n8n.
2. Configure Google Sheets credentials.
3. Configure Google Gemini credentials.
4. Configure LinkedIn and X credentials.
5. Add article data to Google Sheets.
6. Execute the workflow.
